 Flour Mill Management App

A simple flour mill management software built using Python and Tkinter.

This application helps local mills manage orders, calculate processing cost and time, and store order data digitally.

 Why This Software?

Many local mills still use manual methods for order tracking and calculations. This software helps reduce calculation errors, improve order management, and estimate processing time more efficiently.

 Features

- Add grain orders
- Automatic cost calculation
- Processing timer
- JSON data storage
- Simple GUI using Tkinter
- Input validation and error handling

 Technologies Used

- Python
- Tkinter
- JSON
